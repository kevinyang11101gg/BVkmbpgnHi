# 项目简介

## 核心功能
- 爬虫系统：自动抓取大量公开旅游线路和旅游团信息
- 可视化功能：前端交互式登录页面，实现数据展示和用户交互

## 适用场景
- 旅游行业数据采集
- 旅游市场分析
- 旅游线路规划

## 技术特点
- 采用Python进行开发，保证系统的稳定性和高效性
- 爬虫技术高效抓取数据，确保数据量丰富
- 前端可视化交互，提升用户体验
